# Support

Perch is in early development. Support is handled through GitHub until a dedicated product support channel exists.

## Questions

Use GitHub Discussions or open a question issue if Discussions are not enabled.

Good questions include:

- what you tried
- what you expected
- what happened instead
- operating system and runtime versions
- relevant logs without secrets

## Bugs

Use the bug report template. Include reproduction steps and screenshots for UI issues.

## Feature Requests

Use the feature request template. Explain the customer problem before proposing implementation details.

## Security

Do not open public issues with exploit details. Read [SECURITY.md](SECURITY.md).
