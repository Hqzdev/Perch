# Code of Conduct

## Our Pledge

We pledge to make participation in Perch respectful, focused, and productive for everyone.

## Expected Behavior

- Be direct without being hostile.
- Critique ideas, code, and decisions, not people.
- Assume technical discussions should converge on evidence and maintainability.
- Respect different experience levels.
- Keep issues and pull requests focused on the project.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Sexualized language or imagery.
- Deliberate intimidation or sustained disruption.
- Publishing private information without permission.
- Bad-faith arguments that derail project work.

## Enforcement

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code.

## Scope

This code applies to project spaces including GitHub issues, pull requests, discussions, and any project-maintainer communication channel.
